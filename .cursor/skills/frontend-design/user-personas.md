# User Personas & Emotion Mapping

Quick reference for designing for different user groups.

## User Personas

### Gen Z (Born 1997-2012)
**Characteristics:** Digital natives, mobile-first, value authenticity, short attention spans  
**Design:** Vibrant colors, large typography, vertical scroll, fast interactions, peer reviews

### Millennials (Born 1981-1996)
**Characteristics:** Value experiences, research before buying, socially conscious  
**Design:** Muted pastels, clean sans-serif, card-based layout, smooth animations, value-driven content

### Gen X (Born 1965-1980)
**Characteristics:** Independent, value efficiency, skeptical of marketing  
**Design:** Professional colors, familiar typography, clear hierarchy, functional interactions, fact-based content

### Baby Boomers (Born 1946-1964)
**Characteristics:** Detail-oriented, loyal when trusted, value personal service  
**Design:** High contrast, large typography (18px+), simple linear layout, minimal interactions, comprehensive content

## Psychology Checklist

- [ ] **Hick's Law:** No more than 7 choices in navigation
- [ ] **Fitts' Law:** Primary CTAs are large and reachable
- [ ] **Miller's Law:** Content is chunked appropriately
- [ ] **Von Restorff:** CTAs stand out from surroundings
- [ ] **Trust:** Security badges, reviews, policies visible
- [ ] **Emotional:** Design evokes intended feeling
- [ ] **Cognitive Load:** Interface is clean, not overwhelming
- [ ] **Familiar Patterns:** Standard conventions used
- [ ] **Feedback:** All actions have clear responses
- [ ] **Accessibility:** Inclusive for all users
